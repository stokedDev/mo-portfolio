What is JavaScript?

let addition = num => num + num;

JavaScript is one of the main programming languages used for frontend web development.

JS is also one of the main programming languages used for backend web development.

  • JS can be used on the frontend to dynamically change a client-facing web page based on user input.
  • Node.js is a JavaScript framework and runtime environment that allows JS to run outside the web browser.

Variables

var variable = 'A variable declared with the old "var"';
let variable2 = 'A variable declared with the mutable "let"';
const VARIABLE_3 = 'A variable declared with const(short for constant)';

In 2015 two important new JavaScript keywords were introduced: let and const.

Let your go-to be "const"(as it will help you not make bugs), use "let" if you need to(as it can be mutated), and use var only if you need to access and/or modify your variable from anywhere(no matter whether it's in a function or not). Try not to use var because you should be able to accomplish everything you need with "let" and "const".

Const and let variables can be accessed globally like var if they are created on a line alone without being in a function. But, if they are created within a function, they won't exist outside. This means that you can declare two const variables with the same name just as long as they are in separate scopes. They will both render and can have the same or differing values.

  • Variables help make things faster and easier by allowing you to store functions, arrays, objects, calculations, values, and more.

Conditionals

if (height >= 4) {
console.log('You are tall enough to ride this roller coaster.');
}

A conditional is how javascript or any other programming language can use boolean logic to get an answer based on that boolean logic. Boolean logic refers to what determines a value to be true or false.

The following are examples of JS conditionals: if, else if, else, switch statements, ternaries.

These can be used to dynamically solve problems based on possible factors.

DOM Manipulation

const HEADER = document.getElementsByTagName('h1')[0]

When creating or getting elements to use in your JS, always remember to type the keyword "document" first.

To adjust the style of an element, code like this: HEADER.style.fontSize = '1.2rem'

  • All style properties in JS are the same as CSS but they are camel cased instead of using dashes.
  • All style values in JS are declared inside quotes.
