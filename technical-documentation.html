Variables

var variable = 'A variable declared with the old "var"';
let variable2 = 'A variable declared with the mutable "let"';
const VARIABLE_3 = 'A variable declared with const(short for constant)';

In 2015 two important new JavaScript keywords were introduced: let and const.

Let your go-to be "const"(as it will help you not make bugs), use "let" if you need to(as it can be mutated), and use var only if you need to access and/or modify your variable from anywhere(no matter whether it's in a function or not).

Const and let variables can be accessed globally like var if they are created on a line alone without being in a function. But, if they are created within a function, they won't exist outside. This means that you can declare two const variables with the same name just as long as they are in separate scopes.

  • Variables help make things faster and easier by allowing you to store functions, arrays, objects, calculations, values, and more.

"Conditionals", short for "conditional statements" are used to solve problems that require conditions. A Conditional's result may vary just like how people's opinions of something may vary. For example, an apple will be liked by someone if they like the apple's color, and someone else might like that same apple because of it's taste, but then if someone else dislikes the apple completely then you have different results for who likes or dislikes the apple because of different reasons for each person. Here's some pseudo code to demonstrate this logic.

function appleOpinion(person){ if(person likes apples){ return why; } else { return person dislikes apples; } };

The following are examples of JS conditionals: if, else if, else, switch statements.

These can be used to solve problems based on possible factors.

  • The if statement is the most common type of conditional.

DOM Manipulation

const HEADER = document.getElementsByTagName('h1')[0]

When creating or getting elements to use in your JS, always remember to type the keyword "document" first.

To adjust the style of an element, code like this: HEADER.style.fontSize = '1.2rem'

  • All style properties in JS are the same as CSS but they are camel cased instead of using dashes.
  • All style values in JS are declared inside quotes.
