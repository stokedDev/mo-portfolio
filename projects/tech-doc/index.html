What is JavaScript?

let doubleNum = num => num + num;
doubleNum(1) // returns 2
doubleNum(2) // returns 4
doubleNum(3) // returns 6
doubleNum(number) // returns number + number

JavaScript was made for the user-facing part of a website. It was meant to work with HTML and CSS as their lightweight beginner-friendly programming language. JS was meant to be read by the browser engine exclusively. It was meant to be like Java but easier and simpler. JS can be written in a script element in an HTML file, or in a file of its own that is linked to in the HTML file. <script>
/*
JS can be written in a script element which consists of an opening and closing script tag.
*/</script>

JavaScript is one of the main programming languages used for front end web development.

JS(or JavaScript) is also one of the main programming languages used for backend web development.

  • JS can be used on the front end to dynamically change a client-facing web page in response to user input.
  • Node.js is a JavaScript framework and runtime environment that allows JS to run outside the web browser.

How to learn JS

There are many ways to learn JS but here are some of my recommendations.

  • Search on a good search engine ( google, for example ) to find the JavaScript information you need.
  • freeCodeCamp.org will guide you through a curriculum for free.
  • Mozilla Developer Network has exhaustive ( or near-exhaustive ) technical documentation that is likely to be very helpful while learning.
  • One of my favorite ways to learn JS is from the Net Ninja on YouTube .
  • Scrimba , a website with interactive coding tutorials, makes hands-on learning very convenient.
  • Exercism offers completely free mentoring and challenges
  • Learning JavaScript online is likely to be successful if what is learned is put into practice.

Variables

var variable = 'A variable declared with the old "var"';
let variable2 = 'A variable declared with the mutable "let"';
const VARIABLE_3 = 'A variable declared with const(short for constant)';

In 2015 two important new JavaScript keywords were introduced: let and const.

Variables: declared with the "const" and "let" keywords may have global scope, function scope, or block scope; declared with the "const" keyword cannot be modified unless they are arrays; declared with the "let" keyword can be modified; declared with the "var" keyword can be modified, can have global scope, can have function scope, but cannot have block scope. ( A variable with block scope cannot be accessed outside of a loop, an if statement, nor the block scope that it is declared in. )

  • Variables help make things faster and easier by allowing you to store functions, arrays, objects, calculations, values, and more.
  • Variables allow you to reuse one or more lines of code without having to repeat yourself.
  • Descriptive variable naming and efficient use of variables is one of the most key parts of making a JS codebase maintainable.

Conditionals

if (height >= 4) {
console.log('You are tall enough to ride this roller coaster.');
}

A conditional is how JavaScript or any other programming language can use boolean logic to return a result.

Boolean logic refers to what determines a value to be true or false.

The following are examples of JS conditionals: if, else if, else, switch statements, ternaries.

JS conditionals can be used to dynamically solve problems by using boolean logic to return a result.

DOM Manipulation

const HEADER = document
.getElementsByTagName('h1')[0]

The DOM (Document Object Model), allows a JS file to interact with any HTML file that links to the JS file path.

When creating or getting elements to use in your JS, always remember to type the keyword "document" first.

To adjust the style of an element, code like this: HEADER.style.fontSize = '1.2rem'

  • All style properties in JS are the same as CSS but they are camel cased instead of using dashes.
  • All style values in JS are declared inside quotes.
